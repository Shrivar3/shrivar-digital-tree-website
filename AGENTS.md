# Repository guide

- This is a Next.js App Router personal website. Use `npm.cmd` on Windows.
- Public claims and visible copy belong in `src/data/content.ts`; verified external URLs belong in `src/data/links.ts`.
- Never invent project results, company outcomes, dates, contact details, or links.
- Preserve the living-tree navigation model, keyboard focus states, responsive layouts, and reduced-motion behavior.
- Use original SVG/CSS visuals for evidence. Do not add stock project imagery.
- Optional real photos must use the exact filenames documented in `docs/PHOTO_SHOT_LIST.md`.
- Before handoff, run `npm.cmd run typecheck` and `npm.cmd run build`, then check desktop and mobile in a real browser.
